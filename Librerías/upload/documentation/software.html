- Software -

UploadBean is a JAVA JSP/Servlets upload component. Here are the three requirements you need to run it :

• Servlet Engine or Application Server compliant JSP1.1/Servlets2.2 API (or higher).
• JVM 1.2 or higher.
• Browser (Internet Explorer 4.x or higher, Netscape 4.x or higher, Mozilla ...)
• Optionally a database.

UploadBean should run under ANY platform supporting JSP1.1/Servlets 2.2 or J2EE 1.2 (or higher).
